# Automation Policy

## 허용 방향

- 사용자가 직접 로그인한 마이프차 B2B 브라우저 세션 재사용
- 계약상 허용된 범위의 낮은 빈도 업무 자동화
- 화면에 표시되는 업무 데이터를 내부 관리용으로 정리
- 오류 발생 시 자동 중단, 로그 저장, 스크린샷 저장

## 금지 방향

- 로그인 우회
- 캡차 우회
- 차단 회피
- 과도한 반복 요청
- 계약 범위를 벗어난 데이터 재배포
- 개인 또는 민감 정보의 불필요한 저장

## 권장 자동화 흐름

1. 담당자가 PC에서 직접 로그인
2. 자동화 프로그램이 저장된 브라우저 프로필 사용
3. 지정된 메뉴만 이동
4. 필요한 표와 상세 정보만 추출
5. DB 또는 엑셀에 저장
6. 오류 시 즉시 중단하고 스크린샷 저장
