## First-time setup in VS Code

1. Open this repository in VS Code.
2. Install the recommended extensions when VS Code prompts you.
3. Open the Codex sidebar and sign in with your ChatGPT account.
4. Open the Claude Code sidebar and sign in with your Anthropic account.
5. Keep `CLAUDE.md` and `docs/local-setup-codex-windows.md` open as project guidance.
