# Product Plan

## 목표

프랜차이즈 영업단이 마이프차 B2B에서 확인한 정보와 상권 후보 매물을 관리하고, 후보 점수화와 정기 리포트까지 한 화면에서 처리하는 내부 업무 도구를 만든다.

## MVP 범위

- 반응형 대시보드
- 후보 매물 리스트와 상세 보기
- 전략 조건 관리
- 담당자와 상태 관리
- 자동화 실행 로그
- 엑셀/CSV 업로드 기반 데이터 등록
- 주간 리포트 생성

## 4주 계획

1주차: 정적 MVP, 화면 구조, GitHub 저장소, 협업 문서 정리

2주차: Next.js 전환, FastAPI 골격, DB 모델, 샘플 데이터 API

3주차: CSV/엑셀 업로드, 점수 계산, 리포트 생성

4주차: Playwright 자동화, 실행 로그, 오류 스크린샷, 배포 준비

## 주요 데이터

- 후보 매물: 제목, 주소, 지역, 비용, 면적, 업종, 출처, 등록일
- 브랜드: 브랜드명, 카테고리, 권장 면적, 예산 범위, 우선 지역
- 점수: 임대료, 면적, 지역, 업종, 브랜드 적합도, 총점
- 업무: 담당자, 상태, 메모, 다음 액션
