# Franchise Field OS — 네이버 부동산 확장

네이버 부동산에서 보고 있는 매물을 한 번의 클릭으로 후보 매물로 등록하는 Chrome 확장입니다.

## 설치 (Chrome / Edge)

1. Chrome에서 `chrome://extensions` 열기
2. 우상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램 로드** 클릭 → 이 폴더 선택
4. 도구모음에 확장 아이콘 핀 고정

## 사용

1. FastAPI 서버 실행: `powershell -ExecutionPolicy Bypass -File .\scripts\run-api.ps1`
2. 네이버 부동산에서 매물 검색 → 마음에 드는 매물 페이지 열기
3. 확장 아이콘 클릭 → 자동 추출된 폼 검토 → **후보 등록**
4. Franchise Field OS 대시보드 열면 사이드바 "네이버 부동산 세션"에 N건 표시

## 정책 가이드

- **사용자 트리거 단건 등록만 수행**합니다. 자동 페이지 순회/리스트 일괄 수집/봇 감지 우회 없음.
- 추출 정확도는 페이지 구조에 따라 떨어질 수 있습니다 — 등록 전 폼 필드를 항상 확인하세요.
- 네이버 부동산 약관에 위배되는 대량 자동 수집을 위한 도구가 아닙니다.

## 셀렉터/패턴 조정

페이지 구조가 바뀌어 추출이 깨지면 [`selectors.js`](selectors.js)의 `NAVER_REALESTATE_PATTERNS`만 수정하면 됩니다.

## API 서버 위치 변경

기본값은 `http://127.0.0.1:8000`. 다른 호스트/포트면 팝업 하단 **API 서버** 섹션에서 변경.
